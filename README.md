Country Details
==============

This simple webapp has details of 250 countries with respective images of flags of each country. I have used [REST Countries]( https://restcountries.eu/) API to obtain the country data.

## Features
 * Name
 * TopLevelDomain
 * Alpha2Code     
 * Alpha3Code     
 * CallingCodes    
 * Capital       
 * AltSpellings      
 * Region   
 * Subregion
 * Population
 * Latlng
 * Demonym 
 * Area      
 * Gini 
 * Timezones
 * Borders    
 * NativeName    
 * NumericCode
 * Currencies
 * Languages      
 * Translations       
 * Flag
 * RegionalBlocs
 * Cioc
